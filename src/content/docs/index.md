# Amiga x Shaun Docs

This is our shared documentation site for projects we work on together.

The goal is simple:
- keep decisions from disappearing
- document workflows while they are fresh
- give us one place to find how things work
- make it easy for both of us to reuse patterns later

## What belongs here

- project overviews
- setup instructions
- operating patterns
- skills and workflows
- architecture notes
- decisions and tradeoffs
- recurring routines

## Working rule

If we build it, fix it, decide it, or reuse it, it should probably end up here.
